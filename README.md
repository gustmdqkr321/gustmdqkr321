### 👋 소개

> **데이터를 해석 가능한 구조로 만들고, 실제 동작하는 시스템으로 구현하는 개발자**

시계열·형상·정형 데이터를 다루며, 문제를 해석 가능한 구조로 재정의하고 이를 분석과 시스템으로 연결한 경험이 있습니다.
감각과 경험에 의존하던 문제를 **데이터 기준으로 변환**하고, **실제 동작하는 구조로 구현**하는 데 강점이 있습니다.

- 🧩 문제를 단순히 해결하는 게 아니라, **데이터와 구조 기반으로 재정의**하고 하나의 시스템으로 연결합니다.
- 🗣️ 도메인 전문가의 자연어 요구사항을 **데이터 기준과 로직으로 변환**해 동작하는 형태로 구현합니다.
- 🚀 분석에서 끝나지 않고 결과를 **서비스 형태로 확장**하는 것을 지향합니다.
